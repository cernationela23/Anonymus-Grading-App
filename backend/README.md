# Serviciu Backend (Node.js & Express.js)

Acest director este responsabil pentru **Logica de Business** și **Gestiunea Datelor** (Business Logic and Data Management) a aplicației. Acesta găzduiește serverul API **RESTful** construit cu **Node.js** și **Express.js**.

## Tehnologii Cheie
* **Runtime/Framework:** Node.js cu Express.js
* **Baza de Date:** PostgreSQL (Relațională)
* **ORM:** Prisma
* **Autentificare:** JWT (JSON Web Tokens)
* **Autorizare:** Middleware pentru Controlul Accesului Bazat pe Roluri (RBAC)

## Responsabilități Funcționale
* **Implementarea** tuturor punctelor finale (endpoint-urilor) **API RESTful** necesare pentru comunicarea cu frontend-ul.
* **Gestiunea** logicii de selecție aleatorie a juriului și **calculul notei finale** (omiterea notei minime și maxime).
* **Securizarea** rutelor pe baza rolurilor de utilizator (MP, Jurat, Profesor) pentru a aplica permisiunile.
