# Client Frontend (React SPA)

Acest director este responsabil pentru **Stratul de Prezentare al Utilizatorului**. Acesta conține componentele React, gestionează randarea interfeței, interacțiunile utilizatorului, rutarea pe partea de client și comunicarea securizată cu API-ul Backend.

## Tehnologii Cheie
* **Framework:** React.js + React Router (Arhitectură bazată pe componente)
* **Apeluri API:** Axios

## Responsabilități de Arhitectură
* **Implementarea** designului responsive (desktop, mobil) pentru a îndeplini cerințele non-funcționale.
* **Gestiunea** stării componentelor și a fluxului de autentificare pe partea de client.
* **Furnizarea** de interfețe dedicate pentru rolurile Student MP, Jurat și Profesor.
